1. Arithmetic Operators

        +	Addition (Also used for string concatenation.) Left -> Right

        -	Subtraction	Left -> Right

        *	Multiplication	Left -> Right

        /	Division	Left -> Right

        %	Modulo (remainder)	Left -> Right

        -	Unary Minus	Right -> Left (eg : -97.34)
